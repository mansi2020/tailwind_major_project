<h1>HTML Descrption</h1>
nav: Defines a navigation section.<br>
div: Defines a generic container.<br>
a: Defines a hyperlink.<br>
ul: Defines an unordered list.<br>
li: Defines a list item.<br>
i: Defines an icon.<br>

<h1>Tailwind classes Descrption</h1>
top-0: Positions an element at the top of its container.<br>
absolute: Positions an element absolutely within its container.<br>
z-50: Sets the z-index to 50.<br>
w-full: Makes an element take up the full width of its container.<br>
flex: Sets an element to use a flexible layout.<br>
flex-wrap: Allows flex items to wrap onto multiple lines if needed.<br>
items-center: Centers items vertically within a flex container.<br>
justify-between: Distributes content evenly between the start and end of a flex container.<br>
px-6: Adds horizontal padding of 1.5rem (6 units).<br>
py-3: Adds vertical padding of 0.75rem (3 units).<br>
navbar-expand-lg: Custom class (not part of standard Tailwind) that may control the navbar's behavior.<br>
container: Sets a maximum width for the container and centers it.<br>
mx-auto: Horizontally centers the container.<br>
justify-between (inside a nested div): Distributes content evenly between the start and end of a flex container.<br>
font-bold: Makes text bold.<br>
text-lg: Sets the text size to large.<br>
mr-4: Adds right margin of 1rem (4 units).<br>
py-2: Adds vertical padding of 0.5rem (2 units).<br>
text-white: Sets the text color to white.<br>
text-gray-800: Sets the text color to a shade of gray (#333333).<br>
scale-125: Applies a scale transform of 1.25 (increases size by 25%).<br>
text-orange-500: Sets the text color to an orange shade (#FFA500).<br>
